## Project Overview
This is a simple **command-line To-Do List Application** developed in **Python**.  
It allows users to **add, view, edit, delete, and mark tasks as completed**. Tasks are **saved in a JSON file**, so your data persists between sessions.

## Features
- Add tasks with **title, description, and category** (Work, Personal, Urgent)
- View **all tasks** with their status (Pending or Completed)
- Mark tasks as **completed**
- Edit task details
- Delete tasks
- Tasks are stored in a **local JSON file**

## Technologies Used
- Python 3.x  
- JSON file handling  
- Command-line interface
